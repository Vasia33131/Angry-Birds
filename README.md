О проекте
Жанр: Arcade / Physics Puzzle
Движок: Unity
Язык: C#
Платформа: WebGL / PC / Mobile

Управление
Зажать ЛКМ / тач — натянуть рогатку (тяни назад)

Отпустить — выстрел птицей

Мышь / палец — прицеливание (траектория полёта отображается пунктиром)


Физика разрушений (Unity Physics 2D)

Звёзды за уровень (1–3 звезды в зависимости от очков)

Постройки из дерева, камня, стекла, льда
